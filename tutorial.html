FabricView

A new canvas drawing library for Android.

Getting Started

Once you setup your project, navigate to your top-level build file and add the following to your projects repositories

    allprojects { 
        repositories {  
            jcenter() 
            maven { url "https://jitpack.io" } 
        } 
    }

Now go to the project's build.grade file where you want to use FabricView and add the following lines of code to the dependencies 

 compile 'com.github.antwankakki:FabricView:latest'

Now go to the layout xml file and add the following to where you want FabricVew to be added 

 '\u003ccom.agsw.FabricView.FabricView
        android:layout_width=\u0022wrap_content\u0022
        android:layout_height=\u0022wrap_content\u0022
        android:id=\u0022@+id/faricView\u0022
        android:layout_centerHorizontal=\u0022true\u0022
        android:layout_alignParentBottom=\u0022true\u0022 \u005c\u003e'

now in your java code, go to any method you want to use the layout in and add the following code to enable the user to draw 

'// grab a handle to the fabric view
        FabricView fabric = (FabricView) findViewById(R.id.faricView);

        // allow the user to draw on the screen
        fabric.setInteractionMode(FabricView.DRAW_MODE);

        // set the background mode to be a graphing paper mode
        fabric.setBackgroundMode(FabricView.BACKGROUND_STYLE_GRAPH_PAPER); '
